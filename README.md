# Sistema de Vendas Python 🛒

Sistema de vendas desktop com interface gráfica, banco de dados local,
relatórios e gráficos.

## Tecnologias
- Python 3
- Tkinter (interface gráfica)
- SQLite3 (banco de dados)
- Pandas (relatórios)
- Matplotlib + NumPy (gráficos e estatísticas)

## Como rodar
pip install pandas matplotlib numpy
python sistema_vendas.py

## Funcionalidades
- Cadastro de produtos
- Registro de vendas com controle de estoque
- Estatísticas em tempo real
- Relatório via Pandas
- Gráfico de vendas por produto
